# Marzneshin SHM Billing template

Шаблон для связки SHM Billing с Marzneshin

## Use

#### В settings сервера добавить

```
marzneshin {
    link: "https://Marzneshin.link"
    username: "username"
    password: "password"
    services: "1,2,3"
}
```
**services - id services из Marzneshin**
